## How It Works

1. **Build Phase**: `npm run build`
   - Vite builds React client → `dist/public/`
   - esbuild bundles Node server → `dist/index.cjs`

2. **Start Phase**: `npm start` (Vercel runs this)
   - Node.js starts Express server
   - Server serves static files from `dist/public/`
   - Handles API routes and SPA routing

3. **Production**: 
   - Frontend served with aggressive caching
   - API requests routed to Express backend
   - All traffic goes through single Node.js process
